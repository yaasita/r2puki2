# これは何？

redmine <=> pukiwiki <=> markdown
を相互に変換したいなぁとおもって書いてる途中

# インストール
こんな感じで

    NeoBundle 'yaasita/r2puki2'

# 使い方

    # pukiwikiへ
    :TOpukiwiki

    # redmineへ
    :TOredmine
